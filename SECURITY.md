## 2) `SECURITY.md`

Esto queda muy pro, aunque sea un portfolio.

No hace falta algo complejo. Solo indicar:

- que no es un producto crítico
- que si alguien detecta una vulnerabilidad, te contacte de forma responsable
- por dónde reportarlo

### Ejemplo simple

```md
# Security Policy

## Supported Versions

This project is maintained as a personal portfolio and the latest version is the only actively supported one.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

Instead, contact me directly so I can review and address it responsibly.
